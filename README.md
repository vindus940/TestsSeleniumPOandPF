# TestsSeleniumPOandPF
Тесты в формате PageObject и PageFactory

Настоящей целью данного репозитория является закрепление реализации паттернов PageObject и PageFactory. Написание xpatch.

BaseTest - общие настройки для всех тестов - неявные ожидания, расширение окна браузера, закрытие окна после выполнения теста.

GoogleAfterSearch, GoogleBeforeSearch, OpenSitePage - реализация PageObject, где каждая новая странца - отдельный класс в PO.
